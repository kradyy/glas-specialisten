
# Project Structure

This repository contains the following main folders:

- **design/**: This is the current design system and UI/UX work in progress for the project.
- **backend/** (previously `varnamo-glas`): This is the backend service, built with Medusa.js.
- **frontend/** (previously `varnamo-glas-storefront`): This is the e-commerce storefront application.

## Folder Overview

- `design/` — Current design and UI/UX work.
- `backend/` — Medusa.js backend for e-commerce operations.
- `frontend/` — Next.js-based e-commerce frontend.

> **Note:** The folders `varnamo-glas` and `varnamo-glas-storefront` have been renamed to `backend` and `frontend` respectively for clarity.
